Retro Game API for Reinforcement Learning

Posted: Mar 10th, 2022, Edited: Dec 29th, 2022
Abstract

A reinforcement learning simulation API for retro games (e.g., Pokemon Gold), built on OpenAI Gym-Retro package, featuring utility classes (recorder, interactor, and dataset), vision transform classes (random cropping, random convolution, and gaussian noise) and a GUI visualizing observation space and state information.

Install

  1. Before installation, make sure these packages have been installed.
  2. Run in the terminal
git clone git@github.com:henrw/gym-retro-pokemon.git
cd gym-retro-pokemon
pip install -e .

Retro Game Integration

  1. Download Rom of the retro game you want to integrate.
  2. Compile Gym-Retro integration GUI tool.
  3. Integrate the game for a runnable Rom; retrieve memory addresses of variables (e.g., HP, x-y-z, attack, etc.) you want to observe.

             resolution: (W,H), resolution of game to be rendered,
             recorder: poke.utils.recorder or None
        """
  def render(self, mode='human', close=False):
    """Render game simulation GUI.

       Args:
         mode: string, 'human' (viewer) or 'rgb_array' (np.array)
         close: bool, whether window shows up
    """

  def step(self, a):
    """Emulate one simulation step.
      
       Arg:
         a: np.array, action
    """

  def reset(self):
    """Reset emulator to the meta state."""

  def close(self):
    """Close viewer and emulator if exist."""
